# Exemplo de Herança e Casting em Java

Este projeto demonstra conceitos básicos de Programação Orientada a Objetos em Java.

## Conceitos abordados
- Herança
- Polimorfismo
- Casting
- Uso do operador `instanceof`

## Descrição
A classe `Cachorro` herda da classe `Animal`. Um objeto `Cachorro` é criado e tratado como `Animal`.  
Antes de realizar o casting, o operador `instanceof` é utilizado para verificar o tipo do objeto, evitando erros em tempo de execução.

Projeto simples voltado para estudo e prática de Java para iniciantes.
